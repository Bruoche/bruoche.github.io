WekinGame (Code source)

Lors de notre projet académique évalué de 3ème année, nous avons pu créer une application de gestion de Wiki basé sur le jeu vidéo que nous avons nommé "WekinGame". Fonctionnant sur le web via un front-end Réact, appelant lui-même un back-end Java Spring lié à une base de données Mongo DB.
Ce projet nous aura ainsi permit d'apprendre la création de logiciel de bout en bout en autonomie afin d'élaborer et faire évoluer une application informatique tout au long de l'année.
